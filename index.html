GitHub&Hexo搭建个人博客

学号：202000130012

姓名：李佳晨

实验题目：利用云平台搭建个人博客

实验目的：熟悉个人博客系统的搭建

Step1 注册GitHub账号

网页搜索GitHub进入官网并注册账号。

Step2 安装Git

到git官网上下载

Git

下载后会有一个Git bash的命令行工具。安装好后，可以用 git --version 在cmd中查看版本。

Step3 安装NodeJS

Hexo是基于Node JS编写的，Node JS的版本需要不低于10.13，建议使用12.0及以上的版本。Windows选择LTS版本即可。

Node JS

安装完成后，在cmd中输入 node -v 和 npm -v 检查是否安装成功。

Step4 安装Hexo

首选创建一个文件夹blog，然后在文件夹下直接右键git bash打开。 输入命令

npm install -g hexo-cli

完成后使用 hexo -v查看以下版本。没问题后Hexo就安装完成了。
接下来使用命令

hexo init myblog

初始一下Hexo,myblog可以用其他名字代替。然后

cd myblog

进入文件夹，接着输入命令

npm install

最后输入 hexo g和hexo server打开hexo的服务。浏览器中输入localhost:4000就可以看到生成的博客了。

Step5 Github创建个人仓库

这一步需要在Github中完成，以用户名创建一个仓库，后缀是.github.io

Step6 生成SSH添加到GitHub

在git bash中输入命令：

git config --global user.name "yourname"

git config --global user.email "youremail"

这里的 yourname 和 youremail对应Github的用户名和邮箱。然后创建SSH

ssh-keygen -t rsa -C "youremail"

这时候会生成.ssh文件夹。里面的id_rsa是私人的一个密钥，id_rsa.pub是公共密钥。把公钥放在Github上，当链接Github到自己的账户上时，就会根据公钥匹配私钥，顺利匹配后才能上传文件到GitHub上。在Github的setting中，找到 SSH keys的选项，点击 New SSH key，把id_rsa.pub里面的信息复制进去。在gitbash中输入命令

ssh -T git@github.com

检查是否成功。
